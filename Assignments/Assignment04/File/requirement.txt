FSD_MERN_VN – ASSIGNMENT 4 

Build a Complete College Info Portal Website (Frontend Only) 

Objective 
Build a fully functional, responsive College Information Website using React.js. 
Only frontend — no backend. 

Your website must have five pages, dynamic routing, reusable components, and must load 
department data from a local JSON file. 
 
PROJECT REQUIREMENTS 
Pages to Build 
1. Home Page 
Must include: 
• Hero section with banner + tagline 
• “Why this college?” section 
• Cards showing top departments 
• Button links to other pages 
 
2. About Page 
Include: 
• College history 
• Mission & Vision 
• Achievements (timeline or cards) 
 
3. Departments Page 
Show all departments using cards: 
• CSE 
• IT 
• ECE 
• Mechanical 
• Civil 
• MBA

Each department card must show: 
• Name 
• Description (2–3 lines) 
• “View More” button 
 
4. Department Details Page (Dynamic Routing) 
When clicking a department card, open: 
/departments/cse 
/departments/mech 
/departments/it 
Page must show: 
• Full description 
• List of courses offered 
• Faculty list 
• Labs / facilities 
This page must use: 
• dynamic route params 
• data loaded from local JSON 
 
5. Contact Page 
Create a working form with: 
• Name 
• Email 
• Department (dropdown) 
• Message 
On form submit: 
• Show the submitted details on the same page 
• No backend required 
 
FSD_MERN_VN – ASSIGNMENT 4 
Mandatory React Concepts that should be used 
1. React Router 
• Pages 
• Dynamic Routes 
• Navigation bar with active route highlight 
2. Components 
Reusable components such as: 
• <Header /> 
• <Footer /> 
• <DepartmentCard /> 
3. Props 
Pass data to components (e.g., department info to cards). 
4. State Management 
Use useState where required: 
• For form handling 
• For storing selected department data 
5.  useEffect 
Use useEffect to load department data from a local JSON file.
